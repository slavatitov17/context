# Context

Система Context предназначена для автоматического анализа документов и создания диаграмм различных типов на основе текстового описания предметной области.

## Характеристика

Context представляет собой веб-приложение, построенное на базе Next.js и TypeScript, которое обеспечивает автоматический анализ содержимого документов с помощью искусственного интеллекта и создание диаграмм по описанию предметной области.

## Возможности

- Загрузка и обработка документов различных форматов (PDF, DOCX, XLSX, TXT)
- Автоматический анализ содержимого документов с помощью искусственного интеллекта
- Создание диаграмм различных типов по описанию предметной области
- Управление проектами и диаграммами
- Интеллектуальный поиск и генерация ответов на основе загруженных документов

## Технологический стек

- Библиотека React для построения пользовательского интерфейса
- Фреймворк Next.js для создания веб-приложений с серверным рендерингом
- Язык TypeScript для повышения надежности кода
- CSS-фреймворк Tailwind CSS для быстрой стилизации интерфейса
- Модель Mixtral 8x22B от Mistral AI для анализа документов и генерации диаграмм
- API PlantUML для преобразования текстового описания в диаграммы
- База данных IndexedDB для хранения проектов и диаграмм в браузере
- Платформа Vercel для развертывания и хостинга веб-приложения
