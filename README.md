# Context

Система для анализа документов и создания диаграмм различных типов.

## Характеристика

Context представляет собой CASE-средство в формате веб-приложения, которое обеспечивает автоматический анализ содержимого документов пользователя и создание диаграмм по описанию предметной области.

## Возможности

- Загрузка документов различных форматов (PDF, DOCX и другие)
- Анализ и выдача ответов на вопросы по загруженным документам
- Создание и экспорт диаграмм различных типов (UML, ER и другие)
- Создание глоссария и кода (PlantUML) диаграмм различных типов

## Технологический стек

CASE-средство разработано с использованием:

- Библиотеки React для построения пользовательского интерфейса
- Фреймворка Next.js для создания веб-приложений
- Языка TypeScript для повышения надежности кода
- CSS-фреймворка Tailwind CSS для быстрой стилизации интерфейса
- Модели Mixtral 8x22B для анализа документов и генерации диаграмм
- API PlantUML для преобразования текстового описания в диаграммы
- Базы данных IndexedDB для хранения проектов и диаграмм в браузере
- Платформы Vercel для развертывания и хостинга веб-приложения
