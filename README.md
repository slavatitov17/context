# Context

Система Context предназначена для автоматического анализа документов и создания диаграмм различных типов на основе текстового описания предметной области.

## Характеристика

Context представляет собой веб-приложение, построенное на базе Next.js и TypeScript, которое обеспечивает автоматический анализ содержимого документов с помощью искусственного интеллекта и создание диаграмм по описанию предметной области.

## Возможности

- Загрузка документов различных форматов (PDF, DOCX, XLSX, TXT и другие)
- Автоматический анализ содержимого документов и выдача ответов на вопросы по документам (RAG-модель)
- Создание диаграмм различных типов по описанию предметной области (Семейство UML и другие)

## Технологический стек

Система разработана с использованием:

- Библиотеки React для построения пользовательского интерфейса
- Фреймворка Next.js для создания веб-приложений с серверным рендерингом
- Языка TypeScript для повышения надежности кода
- CSS-фреймворка Tailwind CSS для быстрой стилизации интерфейса
- Модели Mixtral 8x22B от Mistral AI для анализа документов и генерации диаграмм
- API PlantUML для преобразования текстового описания в диаграммы
- Базы данных IndexedDB для хранения проектов и диаграмм в браузере
- Платформы Vercel для развертывания и хостинга веб-приложения
